## タイプ別マーケコピー（ベネフィットの核）

各タイプに共通するキー: `empathy`, `problem`, `cause`, `solution`, `future`  
（`app.py` の `MARKETING_COPY` を参照）

### 直感ナビ型（intuition_navi）

- **共感**: 感覚で未来を掴むタイプ
- **問題提起**: 方向は見えているのに確信が持てない
- **原因**: 直感の言語化不足
- **リスク**: 意思決定が遅れ、流れを逃す
- **未来**: 方向性の明確化・迷いを確信へ

### 戦略思考型（strategy_thinker）

- **共感**: 構造で未来を作るタイプ
- **問題提起**: 情報過多で最適解が判断しきれない
- **原因**: 判断軸のズレ
- **リスク**: 時間・エネルギーの浪費
- **未来**: 最適な判断軸の明確化

### 行動突破型（action_breakthrough）

- **共感**: 動いて未来を切り開くタイプ
- **問題提起**: 行動しているのに結果がついてこない
- **原因**: 方向の微妙なズレ
- **リスク**: エネルギー消耗
- **未来**: 最短で結果に繋がる方向の明確化

### 調和リーダー型（harmony_leader）

- **共感**: 関係性の中で力を発揮するタイプ
- **問題提起**: 周囲に合わせすぎて自己の方向が曖昧
- **原因**: 自己軸の弱まり
- **リスク**: 本来の力を発揮できない
- **未来**: 自己軸と他者とのバランスの明確化
