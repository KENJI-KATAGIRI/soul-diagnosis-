# 戦略サマリー（外部 × UTAGE）

## 外部LP（本システム）の役割

- **感情・理解・共感**を生む（診断体験・パーソナライズ結果・マーケコピー）
- **魂タイプ・スコア・一次データ**を保持し、ナビAI・本格レポートへ接続可能
- **世界観・信頼・ストーリー**はここが主戦場

## UTAGE LP（サブLP）の役割

- **登録率・CV**を最大化する短い導線（広告・A/B・リターゲ）
- **シンプルなCTA**と**明確なオファー**（本番ストーリーLPの「圧縮版」）
- 診断なし or 診断後の**回収専用**に振り分け可能

## A/Bテストの狙い

- **見出し・CTA・ボタン配置・ファーストビュー**で登録率を改善
- 変数は**1要素ずつ**、UTAGE側で計測しやすい単位で切る
- 勝ちパターンはサブLPに反映し、感情設計は外部LPに**トーンだけ**還元

## コンバージョン最大化ポイント

1. **診断直後**にメール登録（感情が動いた瞬間に回収）
2. **UTAGEへ送るデータ**にタイプ・スコアを含め、ステップ配信で**セグメント**
3. **本格レポート・講座**は「深い納得」のあと — 登録（UTAGE）と**別レイヤー**で設計
4. **外部＝体験、UTAGE＝行動・配信・決済基盤**の役割分担を崩さない
